    ❓ Aide — Gestion Tournoi de Tarot

    🖥 Installation

    • Mac : ouvrez le fichier .dmg, glissez l'app dans Applications. Au 1er lancement : clic-droit → Ouvrir → Confirmer.
    • Windows : double-cliquez le fichier .exe pour installer.

    👤 Gestion des joueurs

    • Ajouter un joueur : tapez le nom et cliquez Ajouter.
    • Cocher/décocher pour inclure ou exclure un joueur du tournoi.
    • Le nombre de joueurs n'a pas besoin d'être un multiple exact de 4 ou 5. Trois options sont proposées selon le nombre de joueurs :
      • Ajouter des morts (1 à 3) — des places vides complètent les tables.
      • Tables de 5 — créer de 1 à 3 tables de 5 joueurs au lieu de 4.
      • Exclure un joueur — un joueur est exclu par rotation à chaque manche et marque automatiquement 180 points.

    🔄 Tirage au sort

    Le tirage mélange aléatoirement les joueurs et génère les rotations pour toutes les manches. Attention : un nouveau tirage remet les scores à zéro.

    📊 Saisie des scores

    • Sélectionnez la manche dans le menu déroulant.
    • Pour chaque table, saisissez uniquement le score de l'attaquant — l'application calcule automatiquement les scores des défenseurs.
    • Cliquez Valider pour enregistrer les scores de la manche.
    • Les scores sont sauvegardés automatiquement au fur et à mesure.

    🎯 Déroulement d'un tournoi

    1. Joueurs / Tirage — Sélectionnez les joueurs présents, réglez le nombre de manches et de parties par manche, puis lancez le Tirage au sort.
    2. Plan de table — Consultez la répartition des joueurs aux tables pour chaque manche.
    3. Saisie scores — Saisissez uniquement le score des preneurs. L'application calcule le reste automatiquement.
    4. Scores Tournoi — Visualisez le classement en direct avec les totaux.
    5. Classement annuel — Classement cumulé sur l'année.
    6. Récapitulatif — Vue d'ensemble de la soirée (exportable en PDF).

    ⏱ Minuteur

    Lancez un compte à rebours pour chaque manche. Des raccourcis rapides (45, 50, 55, 60 min) sont disponibles. Quand le temps est écoulé, le bouton Minuteur clignote en rouge dans la barre de navigation.

    💾 Sauvegarde / Restauration

    • Sauvegarde : exporte toutes les données (joueurs, scores, classement) dans un fichier JSON.
    • Restauration : importe un fichier de sauvegarde précédemment créé.

    Les données sont aussi sauvegardées automatiquement en local sur votre ordinateur.

    💡 Astuce : Les données sont stockées localement. Pour repartir à zéro, supprimez le dossier de données :

    • Mac : ~/Library/Application Support/gestion-tournoi-tarot/data/
    • Windows : %APPDATA%/gestion-tournoi-tarot/data/
